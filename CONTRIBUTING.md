# Contributing to Simple Interest Calculator

We love receiving contributions from the community and are excited to welcome your help! All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

There are many ways to contribute, including:

-   **🐛 Reporting Bugs**: If you encounter a bug, please open an issue on GitHub and provide a detailed description of the problem.
-   **🛠️ Fixing Bugs**: If you'd like to fix a bug, feel free to create a pull request with your changes.
-   **📚 Improving Documentation**: If you notice any gaps or errors in the documentation, please submit a pull request to make it better.
-   **✨ Suggesting Enhancements**: Have an idea for a new feature? Open an issue to share your thoughts.
-   **💡 Sharing Ideas**: We are open to all ideas that can improve this project.

## Getting Started

1.  **Fork** the repository on GitHub.
2.  **Clone** your fork to your local machine.
3.  Create a new **branch** for your changes.
4.  Make your contributions.
5.  **Push** your changes to your fork.
6.  Open a **Pull Request** from your fork to our main repository.

Thank you for helping make this project better!
